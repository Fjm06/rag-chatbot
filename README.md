# RAG Chatbot

A tiny Retrieval-Augmented Generation (RAG) chatbot built from scratch.

## What it does
Answers questions based on a set of text files using embeddings + a vector database + an LLM.

## Stack
- Python
- ChromaDB (vector database)
- Sentence Transformers (embeddings)
- Ollama or OpenAI (LLM)

## Setup
```bash
python -m venv venv
source venv/Scripts/activate  # Windows
pip install -r requirements.txt
```
